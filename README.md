# Roland's Discord Bot 🤖

Placeholder repo — the real build starts when the VPS is ready.

## Quick Start

```bash
git clone https://github.com/AggrMod/roland-discord-bot.git
cd roland-discord-bot
npm install
cp .env.example .env   # add your bot token
node index.js
```

## Setup

1. Create a bot at [Discord Developer Portal](https://discord.com/developers/applications)
2. Copy your bot token into `.env`
3. Invite the bot to your server using the OAuth2 URL generator
