📘 Solpranos Admin Command Guide

Live command reference aligned with current module-first architecture.

🔐 Verification

  • /verification status — Show verification status, tier, VP, holdings.
  • /verification wallets — List linked wallets.
  • /verification refresh — Re-scan holdings and refresh user data.
  • /verification quick — Start micro-transfer quick verification.

Admin: /verification admin panel, export-user, remove-user, export-wallets, role-config, actions, og-view, og-enable, og-role, og-limit, og-sync, activity-watch-add, activity-watch-remove, activity-watch-list, activity-feed.

🏛️ Governance

  • /governance propose — Create proposal.
  • /governance support — Support draft proposal.
  • /governance vote — Vote yes/no/abstain.

Admin: /governance admin list, cancel, settings.

💰 Treasury

  • /treasury view — Public-safe treasury snapshot.

Admin: /treasury admin status, refresh, enable, disable, set-wallet, set-interval, tx-history.

⚔️ Battle

  • /battle create — Create battle lobby.
  • /battle start — Start your open lobby.
  • /battle cancel — Cancel your open lobby.
  • /battle stats — Show battle stats.

Admin: /battle admin list, force-end, settings.

🎯 Heist

Module is currently disabled by default. Commands exist for future re-enable:

/heist view, /heist signup, /heist status, admin: create, list, cancel.

⚙️ Config

  • /config modules — View module toggles.
  • /config toggle — Enable/disable module.
  • /config status — Runtime status overview.
