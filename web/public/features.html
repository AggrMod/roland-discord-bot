Features

Everything your community needs in one platform.

10 integrated modules designed for Discord-native Web3 community operations.

🔐

Wallet Verification & Roles

Give holders instant access without manual checks. Members connect their wallet, GuildPilot verifies ownership, and roles are assigned automatically based on your rules.

Problem it solves: Manual role assignment, fake holders, scattered verification tools.

  • Solana wallet verification
  • NFT collection roles
  • Token balance roles
  • Trait-based rules
  • Holder tier sync
  • Multi-wallet support
  • OG/WL access
  • Private channel gating
  • Signature + on-chain proof
📜

DAO Governance

Run structured community decisions with full proposal lifecycle: creation, support gathering, voting, and transparent results.

Problem it solves: Informal polls, no voting power controls, no quorum or threshold enforcement.

  • Proposal creation
  • Support phase
  • Voting phase
  • NFT/token voting power
  • Quorum settings
  • Threshold settings
  • Role-based governance
  • Public outcomes
📡

Wallet, NFT & Token Tracking

Monitor wallets, collections, token balances, treasury activity, and ecosystem signals from one dashboard.

Problem it solves: Switching between explorers, missing whale movements, no automated alerts.

  • Wallet tracker
  • NFT collection tracker
  • Token tracker
  • Treasury monitoring
  • Discord alerts
  • Activity logs
  • Multi-wallet support
🎯

Missions & Rewards

Create structured missions, track participation, and reward active members with points, XP, and redeemable vault items.

Problem it solves: Low engagement, no activity incentives, no way to reward loyal holders.

  • Mission templates
  • Solo & co-op modes
  • XP & points
  • Leaderboards
  • Vault store
  • NFT/token/WL rewards
  • Invite contests
  • Rank ladder
🎫

Ticketing & Support

Handle user questions, verification issues, partner requests, and internal workflows in organized private channels.

Problem it solves: DM floods, lost context, no audit trail for support interactions.

  • Ticket creation
  • Category routing
  • Admin assignment
  • Private channels
  • Ticket logs
  • Moderator workflows

Minigames & Battles

Add competitive games and events to keep your server active. Holder-only participation with configurable themes and reward pools.

Problem it solves: Dead servers, no retention loops, no fun reasons to check Discord daily.

  • Battle games
  • Reward pools
  • Custom themes
  • Server-specific settings
  • Holder-only access
  • Engagement events

Engagement & Points

Award points for messages and reactions. Run a points shop with role rewards, codes, and custom prizes.

  • Message points
  • Reaction points
  • Points shop
  • Role rewards
  • Custom prizes
  • Leaderboards

AI Assistant

AI-powered channel monitoring, daily recaps, and intelligent community engagement summaries.

  • Channel monitoring
  • Daily recaps
  • Activity summaries
  • Configurable channels

Admin & Branding

Server-specific configuration, custom branding profiles, self-serve role panels, and role-based admin permissions.
