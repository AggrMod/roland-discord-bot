🚀 Getting Started with Solpranos

1

Join the Discord

Make sure you're a member of the Solpranos Discord server.

2

Link Your Wallet

Use the Wallets section to verify ownership of your Solana wallet containing Solpranos NFTs.

3

Get Your Tier

Your tier is automatically assigned based on the number of NFTs you hold.

4

Participate

Start voting on proposals and joining missions to earn rewards.

💼 Wallet Setup Guide

How to Link a Wallet

  1. Click "Add Wallet" in the Wallets section
  2. Enter your Solana wallet address
  3. Sign the verification message using Phantom, Solflare, or another wallet
  4. Submit the signature to complete verification

Supported Wallets

  • Phantom Wallet
  • Solflare
  • Backpack
  • Any wallet that supports message signing

Setting a Favorite Wallet

Click the star icon next to any linked wallet to set it as your favorite. This wallet will be used for voting power calculations.

📜 Governance Guide

Creating a Proposal

  1. Use the /gov create command in Discord
  2. Provide a clear title and detailed description
  3. Wait for admin approval before voting begins

Voting on Proposals

  1. View active proposals in the Governance section or Discord
  2. Use /gov vote with Yes, No, or Abstain
  3. Your voting power is based on your tier

Voting Power

Each tier has different voting power:

  • Associate: 1 VP
  • Soldier: 2 VP
  • Capo: 3 VP
  • Underboss: 4 VP
  • Don: 5 VP

🔧 Troubleshooting

Wallet verification failed

Issue: Signature verification returns an error

Solution:

  • Make sure you're signing with the same wallet you entered
  • Copy the EXACT verification message provided
  • Try a different wallet app (Phantom, Solflare, etc.)
  • Clear your browser cache and try again

Tier not updating

Issue: Your tier doesn't reflect your NFT count

Solution:

  • Run /tier refresh in Discord
  • Wait a few minutes for the blockchain scan to complete
  • Make sure your NFTs are in a verified wallet

Can't see admin features

Issue: Admin panel not visible

Solution:

  • Only Discord administrators can access the admin panel
  • Log out and log back in to refresh permissions
  • Contact a server administrator if you should have access

Still need help?

Contact support in the Discord server's #support channel.
